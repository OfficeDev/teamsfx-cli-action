# Support

## How to file issues and get help  

This project uses [GitHub Issues](https://github.com/OfficeDev/teamsfx/issues) of TemasFx git repo to track bugs and feature requests. Please search the existing 
issues before filing new issues to avoid duplicates.  For new issues, file your bug or 
feature request as a new Issue.

For help and questions about using this project, please read the [README](https://github.com/OfficeDev/teamsfx-cli-action/blob/main/README.md) first, if you encounter any issues regarding usage, please use [`teams-toolkit` tag on StackOverflow](https://stackoverflow.com/questions/tagged/teams-toolkit) to post your questions.

## Microsoft Support Policy  

Support for this project is limited to the resources listed above.
